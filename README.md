# BOK 트로트 아티스트 팬클럽

트로트 아티스트들의 팬클럽 앱입니다.

## 프로젝트 구조

```
BOK/
├── frontend/     # React 프론트엔드
├── backend/      # Spring Boot 백엔드
├── package.json  # 루트 설정
└── README.md     # 프로젝트 설명
```

## 설치 및 실행

### 전체 설치
```bash
npm run install-all
```

### 프론트엔드만 설치
```bash
npm run install-frontend
```

### 앱 실행
```bash
npm start
```

### 빌드
```bash
npm run build
```

## 기술 스택

### Frontend
- React
- React Router
- CSS3

### Backend
- Spring Boot
- Spring Data JPA
- MySQL
